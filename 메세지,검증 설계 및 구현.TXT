Bean Validation 활용
검증 상황
- 회원가입 할 때
- 로그인할 때

회원가입할 때
- Personal_Id, Password, name -> Not Blank, Not null, @Length(min=, max=)

Length 최솟값과 최대값 지정
- Personal_Id, Password  4, 12
- name 1 50


로그인할 때
- Not Blank, Not null

검증 오류일 경우 나오는 메세지
Not Blank ->  내용을 입력하세요
Not null ->  잘못된 값을 입력했습니다 
length -> 지정된 값의 범위에서 벗어났습니다 최소 ~ 최대 범위 내에서만큼 입력하세요
